# Description

A compendium of my configuration files for MacOS and Linux.

# Features

+ editor
  + __neovim__
+ window manager(s)
  + __yabai__
  + __skhd__ (hotkeys)
+ shell
  + __zsh__
  + __oh-my-zsh__ (plugin manager)
+ terminal
  + __kitty__
  + __tmux__
+ other
  + __git__
  + newsboat

# To install

```sh
$ git clone git@github.com:philliptvo/dotfiles.git
$ cd dotfiles
$ ./setup.sh
```

# Post Installation

To install vim plugins, run following command in vim:

```vim
:PlugInstall
```
